# Victor Ekeator Portfolio

Personal portfolio website for Ekeator Victor Uzoma, built with React and Vite. The site presents product design work, full stack project case studies, resume details, and contact information.

## Stack

- React
- Vite
- React Router
- CSS
- React Icons

## Pages

- Home
- About
- Projects
- Resume
- Contact

## Featured Projects

- Drive Konacuv Transport Platform
- SkillSync Artisan Marketplace
- GazaMarket Marketplace Platform

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run lint checks:

```bash
npm run lint
```

Create a production build:

```bash
npm run build
```
